Second iteration of [Tsoder][tsoder]. Chat bot for [Tsoding][tsoding] streams.

## Quick Start

```console
$ nix-shell
$ stack build
$ stack exec HyperNerd secret.ini
```

### Example of a secret.ini file

```ini
[User]
nick = HyperNerd
password = 12345
channel = Tsoding
```
